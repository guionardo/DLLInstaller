# DLL Installer

Programa para automatizar a instalação/registro de DLL .net


## Modo de operação

DLLINSTALLER.EXE arquivo.dll

## Processo

O programa verifica a existência do programa regasm.exe (https://docs.microsoft.com/pt-br/dotnet/framework/tools/regasm-exe-assembly-registration-tool), e executa o registro da DLL.
